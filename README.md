# Number Game

## Description
A simple Java-based Number Guessing Game where the computer generates a random number between 1 and 100, and the player tries to guess it.

## Features
- Random number generation
- User guess validation
- Feedback for high and low guesses
- Limited number of attempts
- Multiple rounds
- Score calculation

## Technologies Used
- Java
- Scanner Class
- Random Class

## How to Run
1. Compile the program:
   javac NumberGame.java

2. Run the program:
   java NumberGame
